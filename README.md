## Setup Instructions

### Prerequisites

- Docker
- Poetry (for Python)

### Clone the Repository

```sh
https://github.com/abrarmudhir/de-journey.git
cd de-journey
```

### Install Dependencies
- **Python**:
  ```sh
  poetry install
  ```
  
### Running Tests
- **Python**:
    ```sh
    poetry run pytest
    ```
